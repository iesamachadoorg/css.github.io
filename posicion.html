Posicion

Tamaño de un DIV

Un DIV es una caja sin tamaño ni formato predefinido. Es una caja que podemos hacer con ella lo que queramos.

Al DIV, podremos ponerle una imagen de fondo, darle un tamaño y posicionarla en la pantalla donde queramos.

Para dar tamaño a un DIV, usaremos los atributos

  • width
  • height

Ejemplo

#caja {

     width:200px;

     height:300px;

}

Posición de un DIV en la pantalla

Para posicionar un div, hay muchas formas, la más simple es:

Position:Absolute

Esto, ubica nuestro elemento en una esquina de la pantalla y con los parametros de posición, podemos ubicarla pixel a pixel donde queramos. Los parametros de posición son:

  • top
  • bottom
  • left
  • rigth

Con esto, lo moveremos tantos pixeles a un lado u otro.

Ejemplo:

#caja1{

 position:absolute;

top :200px;

left;200px;

}

Para usar este tipo de ubicación indicando el sitio exacto de la pantalla, SIEMPRE deberemos poner la linea

position:absolute;

Cajas sobre Cajas

Las cajas se pueden superponer. Entonces, ¿qué capa está encima?

Para ello, usaremos el parametro z-index con un número. El número más alto, estará encima del resto.

Ejemplo:

#Caja1{

     z-index:1;

}

#caja2 {

    z-index:2;

}

La caja 2, estará encima de la caja 1

Eventos sobre un elemento

Por ejemplo, podemos hacer que cuando ponga el ratón encima de una caja, esta crezca de tamaño o desaparezca.

Podemos tener un css así
